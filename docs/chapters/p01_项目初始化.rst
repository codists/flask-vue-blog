=============================
第一章：项目初始化
=============================

本教程采用自底向上的方法介绍如何从无到有创建一个个人博客系统。本教程重点是告诉你“为什么这样写（why）”，而不是“怎么写（how）”。

Flask-Vue 博客系统前端使用框架 `Vue.js <(https://v3.vuejs.org/>`_，后端使用框架 `Flask <https://flask.palletsprojects.com>`_，数据库使用 关系型数据库 `MySQL <https://www.mysql.com>`_。技术选型的时候为什么选择了 Vue.js，Flask 和 MySQL 呢？因为这三样技术在实际的开发中应用比较普遍，掌握这些技术后有利于在实际的开发中使用。
